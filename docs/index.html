kotoba-lang/office-style

Pure CLJC / EDN StyleIR extraction for PowerPoint packages. It reads deterministic theme colors, fonts, slide size, masters, layouts, and slide parts from .pptx, then emits preview SVG and svgraph-friendly EDN without a JavaScript runtime.

office-style.opcPowerPoint package XML reader.
office-style.styleDeterministic StyleIR extraction.
office-style.previewCompact preview SVG from StyleIR.
office-style.svgraphsvgraph-friendly EDN projection.
(require '[office-style.style :as style]
         '[office-style.preview :as preview]
         '[office-style.svgraph :as svgraph])

(def ir (style/extract-bytes bytes))
(preview/preview-svg ir)
(svgraph/presentation ir)
